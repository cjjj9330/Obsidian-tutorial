# Outliner 插件使用演示

Outliner插件可以增强Obsidian中的列表体验，提供更多列表操作和格式化功能。

## 基本列表功能

- 第一级列表项
	- 第二级列表项
		- 第三级列表项
	- 另一个第二级列表项
- 第一级列表项2

## Outliner主要功能演示

### 1. 折叠/展开列表

- 可折叠的列表项 (使用左侧的小箭头)
	- 子项1
	- 子项2
	- 子项3

### 2. 列表项移动

使用快捷键或右键菜单可以：
- 向上/向下移动列表项 (Ctrl+Shift+方向键)
- 提升/降低列表级别 (Tab/Shift+Tab)
	- 这个列表项可以提升（使用Shift+Tab）
- 这个列表项可以降级（使用Tab）

### 3. 清单功能

- [ ] 未完成任务
- [x] 已完成任务
	- [ ] 子任务1
	- [x] 子任务2

### 4. 样式设置

Outliner允许自定义：
- 列表缩进
- 行间距
- 项目符号样式

### 5. 快捷操作

- 在列表项末尾按Enter会创建新的同级列表项
- 在空列表项按Enter会退出列表
- 使用Ctrl+Enter可在列表项内创建新行而不创建新列表项

### 使用技巧

1. 使用**拖放**功能重组列表项
2. 使用**右键菜单**访问更多列表操作
3. 在设置中启用**线条**连接相关列表项
4. 使用**Tab**和**Shift+Tab**调整缩进层级

> 提示：Outliner设置可在Obsidian设置面板中的"插件选项"下找到。
