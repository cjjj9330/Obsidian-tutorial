## 🚀 快速开始

1. **安装插件**：确保已安装并启用Kanban插件
2. **查看说明**：先阅读[[Kanban插件功能说明]]了解基本概念
3. **浏览示例**：根据您的使用场景选择相关示例查看
4. **动手实践**：复制示例内容，创建您自己的看板

## 💡 使用建议

### 新手用户
- 从简单的个人任务管理开始
- 先掌握基本的拖拽和编辑操作
- 逐步添加标签、日期等高级功能

### 进阶用户
- 探索与其他插件的集成功能
- 自定义标签颜色和看板样式
- 建立适合团队的工作流规范

### 团队使用
- 制定统一的标签和命名规范
- 定期同步和更新任务状态
- 利用归档功能进行项目复盘

## 🛠️ 自定义提示

每个示例文件都可以作为模板使用：
1. 复制文件内容到新的Markdown文件
2. 在文件开头添加`kanban-plugin: basic`标识
3. 根据实际需求修改列表名称和内容
4. 保存后即可在Obsidian中以看板形式查看
