# 数据导入示例

本文档展示了如何在 Obsidian Charts 插件中从外部数据源导入数据，包括 CSV 文件和 Dataview 查询。

## Obsidian Charts 插件简介

Obsidian Charts 插件使用简单的 YAML 语法创建交互式图表，基于 Chart.js 库。它支持两种使用方式：
1. **chart 代码块**：用于静态数据
2. **dataviewjs + window.renderChart API**：用于动态数据和Dataview集成

### 基本语法结构

**静态图表（chart 代码块）：**
```chart
type: line
labels: [Jan, Feb, Mar, Apr, May]
series:
  - title: Sales
    data: [10, 20, 15, 25, 30]
  - title: Costs  
    data: [5, 10, 8, 12, 15]
```

**动态图表（dataviewjs API）：**
```dataviewjs
const chartData = {
    type: 'bar', // 图表类型：'line', 'bar', 'pie', 'doughnut', 'radar' 等
    data: {
        labels: ['标签1', '标签2', '标签3'], // 标签数组
        datasets: [{
            label: '数据系列名称',
            data: [10, 20, 30], // 数据数组
            backgroundColor: ['rgba(255, 99, 132, 0.8)'], // 颜色数组
            borderColor: ['rgba(255, 99, 132, 1)'], // 边框颜色数组
            borderWidth: 1
        }]
    }
};

window.renderChart(chartData, this.container);
```

## 从 CSV 文件导入数据

### 重要说明

根据官方文档，Charts插件**不支持直接引用CSV文件**。要使用CSV数据，有以下几种方法：

1. **将CSV数据转换为Markdown表格**（推荐）
2. **使用Dataview插件读取CSV文件**
3. **手动将数据输入到chart代码块中**

### 创建 CSV 文件

首先，创建一个 CSV 文件 `月度销售数据.csv`：

```csv
月份,销售额,成本,利润
一月,12000,7000,5000
二月,19000,11000,8000
三月,15000,9000,6000
四月,25000,14000,11000
五月,22000,12500,9500
六月,30000,16000,14000
```

### 使用 CSV 数据创建图表

**方法1：从表格创建图表（推荐）**

首先创建一个Markdown表格：

|       | 一月    | 二月    | 三月    | 四月    | 五月    | 六月    |
| ----- | ----- | ----- | ----- | ----- | ----- | ----- |
| 销售额 | 12000 | 19000 | 15000 | 25000 | 22000 | 30000 |
| 成本  | 7000  | 11000 | 9000  | 14000 | 12500 | 16000 |
| 利润  | 5000  | 8000  | 6000  | 11000 | 9500  | 14000 |
^sales-table

然后使用表格ID创建图表：

```chart
type: bar
id: sales-table
layout: rows
width: 80%
beginAtZero: true
```

**方法2：使用 Dataview 读取 CSV 文件**

```dataviewjs
// 读取CSV文件数据
try {
    const csvData = await dv.io.csv("月度销售数据.csv");
    
    if (csvData && csvData.length > 0) {
        const labels = csvData.map(row => row["月份"]);
        const salesData = csvData.map(row => parseInt(row["销售额"]));
        const costData = csvData.map(row => parseInt(row["成本"]));
        const profitData = csvData.map(row => parseInt(row["利润"]));

        const chartData = {
            type: 'line',
            data: {
                labels: labels,
                datasets: [{
                    label: '销售额',
                    data: salesData,
                    borderColor: 'rgb(75, 192, 192)',
                    backgroundColor: 'rgba(75, 192, 192, 0.2)',
                    tension: 0.1
                }, {
                    label: '成本',
                    data: costData,
                    borderColor: 'rgb(255, 99, 132)',
                    backgroundColor: 'rgba(255, 99, 132, 0.2)',
                    tension: 0.1
                }, {
                    label: '利润',
                    data: profitData,
                    borderColor: 'rgb(54, 162, 235)',
                    backgroundColor: 'rgba(54, 162, 235, 0.2)',
                    tension: 0.1
                }]
            },
            options: {
                responsive: true,
                plugins: {
                    title: {
                        display: true,
                        text: '从CSV文件导入的销售数据'
                    }
                },
                scales: {
                    y: {
                        beginAtZero: true,
                        title: {
                            display: true,
                            text: '金额 (元)'
                        }
                    }
                }
            }
        };

        window.renderChart(chartData, this.container);
    } else {
        dv.paragraph("❌ 无法读取CSV文件数据");
    }
} catch (error) {
    dv.paragraph(`❌ 错误: ${error.message}`);
}
```

## 与 Dataview 集成

### 从当前页面元数据创建图表

假设当前页面有以下元数据：
```yaml
---
test: 第一次测试
mark: 6
---
```

```dataviewjs
const data = dv.current();

// 调试信息：显示读取到的数据
dv.paragraph(`**调试信息：**`);
dv.paragraph(`- test: ${data.test}`);
dv.paragraph(`- mark: ${data.mark} (类型: ${typeof data.mark})`);
dv.paragraph(`- 所有字段: ${Object.keys(data).join(', ')}`);

// 确保数据是数字类型
const markValue = typeof data.mark === 'string' ? parseFloat(data.mark) : data.mark;
const testName = data.test || '未知测试';

if (markValue !== undefined && !isNaN(markValue)) {
    const chartData = {
        type: 'bar',
        data: {
            labels: [testName],
            datasets: [{
                label: 'Grades',
                data: [markValue],
                backgroundColor: ['rgba(255, 99, 132, 0.8)'],
                borderColor: ['rgba(255, 99, 132, 1)'],
                borderWidth: 1
            }]
        },
        options: {
            responsive: true,
            plugins: {
                title: {
                    display: true,
                    text: '当前页面测试成绩'
                }
            },
            scales: {
                y: {
                    beginAtZero: true,
                    max: 10,
                    title: {
                        display: true,
                        text: '分数'
                    }
                }
            }
        }
    };

    window.renderChart(chartData, this.container);
} else {
    dv.paragraph("❌ 无法读取到有效的mark数据");
}
```

### 习惯追踪示例

```dataviewjs
// 假设你有每日笔记，包含习惯追踪数据
const pages = dv.pages('"日记"').where(p => p.运动 !== undefined).sort(p => p.file.name).limit(7);

if (pages.length > 0) {
    const labels = pages.map(p => p.file.name.split('-').slice(-2).join('-')); // 提取日期
    const exerciseData = pages.map(p => p.运动 ? 1 : 0);
```

### 项目任务统计

```dataviewjs
// 统计不同项目的任务完成情况
const projectPages = dv.pages().where(p => p.file.tasks && p.file.tasks.length > 0);

if (projectPages.length > 0) {
    const projectStats = [];
    
    for (let page of projectPages.limit(10)) {
        const tasks = page.file.tasks;
        const completed = tasks.filter(t => t.completed).length;
        const total = tasks.length;
        
        if (total >= 3) { // 只统计有至少3个任务的页面
            projectStats.push({
                name: page.file.name.length > 15 ? 
                    page.file.name.substring(0, 12) + '...' : 
                    page.file.name,
                completed: completed,
                total: total
            });
        }
    }

    if (projectStats.length > 0) {
        const labels = projectStats.map(p => p.name);
        const completedData = projectStats.map(p => p.completed);
        const remainingData = projectStats.map(p => p.total - p.completed);

        const chartData = {
            type: 'bar',
            data: {
                labels: labels,
                datasets: [{
                    label: '已完成',
                    data: completedData,
                    backgroundColor: 'rgba(75, 192, 192, 0.8)'
                }, {
                    label: '未完成',
                    data: remainingData,
                    backgroundColor: 'rgba(255, 99, 132, 0.8)'
                }]
            },
            options: {
                responsive: true,
                plugins: {
                    title: {
                        display: true,
                        text: '项目任务完成情况'
                    }
                },
                scales: {
                    x: {
                        stacked: true
                    },
                    y: {
                        stacked: true,
                        beginAtZero: true,
                        title: {
                            display: true,
                            text: '任务数量'
                        }
                    }
                }
            }
        };

        window.renderChart(chartData, this.container);
    } else {
        dv.paragraph("❌ 没有找到符合条件的项目数据");
    }
} else {
    dv.paragraph("❌ 没有找到包含任务的页面");
}
```

## 使用技巧

### 1. 数据源选项
- **静态数据**：使用 `chart` 代码块
- **CSV 文件**：使用 `dv.io.csv()` 读取
- **Dataview 查询**：使用 `dv.pages()` 等查询方法
- **当前页面数据**：使用 `dv.current()`

### 2. 错误处理
```dataviewjs
try {
    const pages = dv.pages().where(p => p.someField);
    
    if (pages.length === 0) {
        dv.paragraph("❌ 没有找到数据");
        return;
    }
    
    // 图表代码...
    
} catch (error) {
    dv.paragraph(`❌ 错误: ${error.message}`);
}
```

### 3. 性能优化
- 使用 `.limit()` 限制查询结果数量
- 对复杂查询结果进行缓存
- 避免在大量页面上执行复杂操作

## 故障排除

### 常见问题

1. **图表不显示**
   - 确保 Charts 插件已启用
   - 检查 Dataview 插件是否已启用（使用 dataviewjs 时）
   - 验证数据格式是否正确

2. **CSV 文件读取失败**
   - 确认文件路径正确（相对于 vault 根目录）
   - 检查文件编码（使用 UTF-8）
   - 确认文件存在且可访问

3. **Dataview 查询错误**
   - 检查查询语法是否正确
   - 确认被查询的页面或字段存在
   - 使用 try-catch 进行错误处理

### 调试建议

- 使用 `dv.paragraph()` 输出调试信息
- 检查浏览器控制台的错误信息
- 从简单示例开始，逐步增加复杂度
