# 实际应用场景

本文档展示了 Obsidian Charts 插件在实际笔记场景中的应用示例，包括习惯追踪、项目管理、学习进度和个人财务等。

## 习惯追踪

使用图表可视化你的习惯养成情况，帮助你保持动力和了解进度。

### 每日习惯完成情况

```chart
type: bar
labels: [周一, 周二, 周三, 周四, 周五, 周六, 周日]
series:
  - title: 早起
    data: [1, 1, 0, 1, 1, 0, 0]
  - title: 冥想
    data: [1, 1, 1, 0, 1, 1, 1]
  - title: 阅读
    data: [0, 1, 1, 1, 0, 1, 1]
  - title: 运动
    data: [1, 0, 1, 1, 1, 0, 0]
options:
  title:
    display: true
    text: 本周习惯完成情况
  scales:
    y:
      min: 0
      max: 1
      ticks:
        stepSize: 1
        callback: "function(value) { return value === 0 ? '未完成' : '完成'; }"
```

### 习惯连续天数

```chart
type: line
labels: [1月, 2月, 3月, 4月, 5月, 6月]
series:
  - title: 早起连续天数
    data: [5, 12, 8, 15, 21, 18]
  - title: 冥想连续天数
    data: [3, 7, 14, 20, 25, 30]
  - title: 阅读连续天数
    data: [10, 5, 8, 12, 15, 20]
options:
  title:
    display: true
    text: 习惯连续天数变化
  elements:
    line:
      tension: 0.4
```

## 项目管理

使用图表跟踪项目进度、资源分配和任务完成情况。

### 项目进度跟踪

```chart
type: bar
labels: [需求分析, 设计, 开发, 测试, 部署]
series:
  - title: 计划工时
    data: [40, 80, 160, 60, 20]
  - title: 实际工时
    data: [35, 95, 140, 80, 15]
  - title: 完成百分比
    type: line
    data: [100, 100, 75, 50, 0]
    yAxisID: percentage
options:
  title:
    display: true
    text: 项目进度跟踪
  scales:
    y:
      title:
        display: true
        text: 工时
    percentage:
      position: right
      min: 0
      max: 100
      title:
        display: true
        text: 完成百分比
      grid:
        drawOnChartArea: false
```

### 团队任务分配

```chart
type: pie
labels: [张三, 李四, 王五, 赵六, 钱七]
series:
  - data: [12, 8, 15, 10, 5]
options:
  title:
    display: true
    text: 团队任务分配
  plugins:
    tooltip:
      callbacks:
        label: "function(context) { return context.label + ': ' + context.raw + ' 个任务'; }"
```

## 结合 Dataview 的高级应用

如果你同时安装了 Dataview 插件，可以实现更强大的图表功能。以下是一个结合 Dataview 的示例：

### 从笔记元数据生成图表

假设你有一系列带有以下前置元数据的笔记：

```yaml
---
date: 2025-06-01
mood: 8
sleep: 7
productivity: 9
exercise: true
meditation: true
---
```

你可以使用 Dataview 查询这些数据并生成图表：

```chart
type: line
data:
  dataview:
    query: "TABLE date, mood, sleep, productivity FROM \"日记\" WHERE date >= date(2025-06-01) AND date <= date(2025-06-30) SORT date ASC"
    labels: "date"
    series:
      - "mood"
      - "sleep"
      - "productivity"
options:
  title:
    display: true
    text: 六月份状态追踪
  scales:
    y:
      min: 0
      max: 10
      title:
        display: true
        text: 评分 (0-10)
```

## 与其他插件结合使用

Charts 插件可以与其他 Obsidian 插件结合使用，创造更丰富的笔记体验。

### 与 Templater 结合

使用 Templater 插件可以自动生成图表代码：

```javascript
<%*
// 获取过去7天的日期
let dates = [];
for(let i=6; i>=0; i--) {
  let d = new Date();
  d.setDate(d.getDate() - i);
  dates.push(d.toISOString().split('T')[0]);
}

// 生成随机数据
let data1 = Array(7).fill().map(() => Math.floor(Math.random() * 10));
let data2 = Array(7).fill().map(() => Math.floor(Math.random() * 10));

// 生成图表代码
let chartCode = `\`\`\`chart
type: line
labels: [${dates.map(d => `"${d}"`).join(', ')}]
series:
  - title: 数据1
    data: [${data1.join(', ')}]
  - title: 数据2
    data: [${data2.join(', ')}]
options:
  title:
    display: true
    text: 自动生成的图表
\`\`\``;

tR += chartCode;
%>
```

### 与 Dataview 和 MetaEdit 结合

结合 Dataview 和 MetaEdit 插件，可以创建交互式仪表板：

1. 使用 MetaEdit 更新笔记元数据
2. 使用 Dataview 查询更新后的数据
3. 使用 Charts 可视化查询结果

这种组合可以创建强大的个人知识管理和数据分析工作流。
