# Multi Select

| 属性 | 值 |
|------|----|
| 标识符 | `multiSelect` |
| 允许在代码块中使用 | `true` |
| 允许内联使用 | `false` |

## 描述

多选字段，允许选择多个选项。

## 允许的参数

以下输入字段参数可以用于 `multiSelect` 输入字段：

- **class** - 为字段添加自定义CSS类名。
- **defaultValue** - 设置字段的默认值。
- **title** - 为字段设置标题。
- **showcase** - 在文档中展示字段示例。
- **option** - 为选择类字段添加选项。
- **allowOther** - 允许用户输入其他选项。
